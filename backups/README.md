# 📦 Carpeta de Backups

Esta carpeta está destinada para almacenar archivos de respaldo del proyecto.

## Uso

- **Archivos `.backup`**: Copias de seguridad de archivos Python
- **Archivos `.zip/.tar.gz`**: Compresiones de versiones anteriores
- **Archivos obsoletos**: Código que ya no se usa pero se conserva por referencia

## Mantenimiento

Se recomienda limpiar esta carpeta periódicamente para evitar acumulación innecesaria de archivos.

## Nota

Los archivos en esta carpeta están excluidos del control de versiones Git (ver `.gitignore`).
