# Modelos YOLO

Modelos pre-entrenados de detección de objetos.

## Contenido

- **yolov8n.pt** (6.3 MB) - Modelo nano (más rápido, menos preciso)
- **yolov8m.pt** (50 MB) - Modelo medium (balanceado) ⭐ Usado por defecto

## Uso

Los modelos son cargados automáticamente por `hardware/camera_service.py`.
